# Notre Budget — Application PWA

Application de gestion budgétaire pour couple. Interface mobile-first, données sauvegardées localement sur chaque appareil.

---

## Déploiement sur Vercel (15 minutes, 100% gratuit)

### Étape 1 — Créer un compte GitHub
1. Aller sur [github.com](https://github.com)
2. Cliquer **Sign up** et créer un compte gratuit
3. Confirmer ton adresse e-mail

### Étape 2 — Créer un dépôt et uploader les fichiers
1. Une fois connecté, cliquer sur le **+** en haut à droite → **New repository**
2. Nommer le dépôt : `budget-couple`
3. Laisser en **Public** (nécessaire pour Vercel gratuit)
4. Cliquer **Create repository**
5. Sur la page du dépôt, cliquer **uploading an existing file**
6. Glisser-déposer **tous les fichiers** du dossier `budget-app` (en conservant la structure de dossiers)
7. Cliquer **Commit changes**

### Étape 3 — Déployer sur Vercel
1. Aller sur [vercel.com](https://vercel.com)
2. Cliquer **Sign Up** → choisir **Continue with GitHub**
3. Autoriser Vercel à accéder à GitHub
4. Cliquer **Add New Project**
5. Sélectionner le dépôt `budget-couple`
6. Vercel détecte automatiquement que c'est une app React — laisser les paramètres par défaut
7. Cliquer **Deploy**
8. Attendre ~2 minutes → Vercel génère une URL du type `budget-couple-xxx.vercel.app`

### Étape 4 — Installer sur vos téléphones

**Sur iPhone (Safari) :**
1. Ouvrir l'URL dans Safari
2. Appuyer sur l'icône **Partager** (carré avec flèche vers le haut)
3. Défiler et appuyer sur **Sur l'écran d'accueil**
4. Nommer l'app **Budget** → **Ajouter**

**Sur Android (Chrome) :**
1. Ouvrir l'URL dans Chrome
2. Appuyer sur les **trois points** en haut à droite
3. Appuyer sur **Ajouter à l'écran d'accueil**
4. Confirmer

---

## Mettre à jour l'app
Pour ajouter des catégories ou modifier l'app :
1. Me demander les modifications → je fournis les fichiers mis à jour
2. Remplacer les fichiers dans le dépôt GitHub
3. Vercel redéploie automatiquement en quelques minutes

## Notes
- Les données sont sauvegardées **localement** sur chaque téléphone (pas de compte, pas de serveur)
- Si vous effacez le cache de votre navigateur, les données sont perdues
- La synchronisation entre vos deux téléphones est prévue dans une version future
